# Heartbeat checklist
# Rook checks these items periodically (default: every 60 min during active hours).
# If nothing needs attention, Rook stays silent.
# Edit this file to customize what Rook monitors.

## Priority checks (every heartbeat)
- Are there unread emails that might need a response?
- Are there calendar events in the next 2 hours I should prepare for?
- Did any scheduled task fail since last check?

## Low-priority checks (only flag if notable)
- Weather: is there a significant change (rain, storm, extreme temp)?

## How NOT to be proactive
- Don't send updates about things that haven't changed
- Don't remind about things the user said they'll handle later
- Don't be proactive about anything that costs money without asking
- Max 3 proactive messages per day
- Active hours only: 7:00 — 22:00
